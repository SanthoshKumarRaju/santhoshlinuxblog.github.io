Linux Basic Commands

A beginner-friendly guide

1. pwd (Print Working Directory)

Usage: pwd

This command displays the current directory you are in.

2. ls (List)

Usage: ls

Lists files and directories in the current directory.

  • ls -l → detailed view
  • ls -a → show hidden files

3. cd (Change Directory)

Usage: cd directory_name

Moves you from one directory to another.

4. mkdir (Make Directory)

Usage: mkdir folder_name

Creates a new directory.

5. rmdir (Remove Directory)

Usage: rmdir folder_name

Removes an empty directory.

6. touch

Usage: touch file.txt

Creates a new empty file or updates a file’s timestamp.

7. cat

Usage: cat file.txt

Displays the contents of a file.

8. cp (Copy)

Usage: cp source.txt destination.txt

Copies files or directories.

9. mv (Move)

Usage: mv old.txt new.txt

Moves or renames files/directories.

10. rm (Remove)

Usage: rm file.txt

Deletes files or directories (use carefully!).

11. man (Manual)

Usage: man command

Shows the manual/help for a command.

12. echo

Usage: echo "Hello World"

Prints text to the terminal.

13. clear

Usage: clear

Clears the terminal screen.

14. exit

Usage: exit

Closes the terminal session.
