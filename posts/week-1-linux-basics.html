3) Deploying Nginx + Static React App

  1. Install Nginx

    First update apt repos and install the Nginx web server.

    $ sudo apt update
$ sudo apt install -y nginx
  2. Build React app

    Compile your React project into static files using npm run build.

    $ npm run build
  3. Copy build to web root

    Move generated build files into the Nginx default web directory.

    $ sudo cp -r build/* /var/www/html/
  4. Check status & reload

    Ensure Nginx is running, then reload to serve the new files.

    $ systemctl status nginx
$ sudo systemctl reload nginx
  5. Verify

    Check HTTP response headers with curl.

    $ curl -I http://localhost
